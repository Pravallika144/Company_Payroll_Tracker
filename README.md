# Company Payroll Tracker

## Description

The Company Payroll Tracker is a web-based application developed using **HTML, CSS, and JavaScript** to simplify employee payroll management. It allows users to add, edit, delete, and search employee records while automatically calculating salary components such as HRA, DA, tax, and net salary. The application also provides payroll statistics, including total employees, total payroll, average salary, highest salary, and lowest salary. Employee data is stored using the browser's Local Storage, ensuring records remain available even after refreshing the page.

## How It Works

The user enters an employee's ID, name, department, and basic salary through the input form. After clicking **Add Employee**, the application validates the input, calculates HRA (20% of the basic salary), DA (10% of the basic salary), gross salary, tax (5% of the gross salary), and the final net salary. The employee details are then displayed in a table and saved in Local Storage. Users can edit or delete existing records, search employees by ID or name, clear all records, download the payroll report as a CSV file, or print the report. The summary section updates automatically to display the total number of employees, total payroll, average salary, highest salary, and lowest salary.

## Features

* Add new employee records
* Edit and delete employee details
* Search employees by ID or name
* Automatic salary calculation (HRA, DA, Tax, Net Salary)
* Display payroll summary statistics
* Store employee records using Local Storage
* Download payroll report as a CSV file
* Print payroll report
* Clear all employee records

## Inputs

* Employee ID
* Employee Name
* Department
* Basic Salary

## Outputs

* Employee payroll table
* HRA, DA, Tax, and Net Salary calculations
* Total Employees
* Total Payroll
* Average Salary
* Highest Salary
* Lowest Salary
* Downloadable CSV payroll report
